efTwo (F2) - Advanced find on Page is installed.
Default shortcut keys
tap 'F' twice, F2, CTRL+SHIFT+F or CTRL+SHIFT+F
Find all forms of a word
for example searching for 'replacing' will find you 'replace', 'replaced', ...
Highlight Google search terms
if you were referred from Google or other search engine, efTwo will automatically highlight your searched terms
Options
go the options page.
